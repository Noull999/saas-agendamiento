⏳ PENDIENTE (USUARIO)
═════════════════════════════════════════════════════════════════════════════

1️⃣ INSTALAR POSTGRESQL
  
  📥 Descargar desde: https://www.postgresql.org/download/
  
  Luego verificar:
  $ psql --version
  
  Y ejecutar:
  $ cd backend && npm run migrate

2️⃣ OBTENER API KEYS (Instrucciones en GET_API_KEYS.md)

  ⭐ STRIPE (Pagos)
    - Account: https://dashboard.stripe.com
    - Crear 2 productos (Pro $19.990, Business $34.990 CLP)
    - Copiar: STRIPE_SECRET_KEY, STRIPE_WEBHOOK_SECRET
    - Copiar: STRIPE_PRICE_PRO, STRIPE_PRICE_BUSINESS
    
  📱 TWILIO (WhatsApp Reminders)
    - Account: https://www.twilio.com
    - Copiar: TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN
    - Obtener: TWILIO_WHATSAPP_FROM
    
  🔔 SENTRY (Opcional - Error Tracking)
    - Account: https://sentry.io
    - Copiar: SENTRY_DSN
    
  📧 SMTP (Opcional - Email Recovery)
    - Gmail / SendGrid / Mailgun
    - Copiar: SMTP_HOST, SMTP_USER, SMTP_PASS

3️⃣ ACTUALIZAR .env EN BACKEND

  STRIPE_SECRET_KEY=sk_test_xxx
  STRIPE_WEBHOOK_SECRET=whsec_xxx
  STRIPE_PRICE_PRO=price_xxx
  STRIPE_PRICE_BUSINESS=price_xxx
  TWILIO_ACCOUNT_SID=ACxxx
  TWILIO_AUTH_TOKEN=xxx
  TWILIO_WHATSAPP_FROM=+1415523888
  SENTRY_DSN=https://xxx@ooo.ingest.sentry.io/yyy
  SMTP_HOST=smtp.gmail.com
  SMTP_USER=email@gmail.com
  SMTP_PASS=app_password

4️⃣ DEPLOYAR

  Opción A: Vercel (Recomendado)
    $ git push
    → Conectar en https://vercel.com
    → Setear environment variables
    
  Opción B: Docker + Railway/Render
    → Pushear + deploying automático
