SaaS de Agendamiento para Salud

Plataforma multi-tenant especializada en LATAM con encriptación HIPAA

Documentación de Producto • Mayo 2026

📋 Resumen Ejecutivo

Plataforma SaaS multi-tenant para gestión de citas médicas, consultorios, clínicas y centros de bienestar. Sistema completo de agendamiento, historial de pacientes, gestión de consultas médicas, prescripciones y recordatorios automáticos vía WhatsApp.

Objetivo: Simplificar la administración de citas médicas, aumentar ocupación de consultorios y mejorar experiencia de pacientes con tecnología segura y confiable.

2. Especificaciones Técnicas

Stack Tecnológico

Backend

  • Node.js 18+ + Express 4
  • SQLite 3 (local/servidor)
  • Twilio API (WhatsApp)
  • Stripe API (pagos)
  • Nodemailer (correo)
  • Jest (32 tests)

Frontend

  • React 19 + Vite
  • React Router v7
  • Tailwind CSS 4
  • Recharts (gráficos)
  • Axios (HTTP)
  • Playwright (e2e)

Rendimiento & Escalabilidad

📦 Bundle Frontend: 250 KB gzip (code-split en 8 páginas lazy-loaded)
⚡ Latencia API: <100ms promedio con caché inteligente
👥 Concurrencia: 100+ usuarios simultáneos (SQLite WAL mode, escalable a PostgreSQL)
🔄 Multi-Tenant: Cada negocio es tenant independiente con datos aislados

Características Técnicas

  • 50+ API Endpoints: Documentados con OpenAPI ready
  • Trabajo en Segundo Plano: Cron job cada 30 min para recordatorios
  • Webhooks Stripe: Activación/cancelación automática de planes
  • Sincronización Offline: Funciona sin conexión, sincroniza después
  • Búsqueda & Filtrado: Pacientes por nombre/RUT, citas por fecha/estado
